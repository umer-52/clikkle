The **Clikkle Assistant** is an AI-powered tool engineered to augment Clikkle-related tasks with technical precision. It operates on a foundation of training data sourced from the Clikkle documentation website, enabling it to furnish insights into Clikkle's features, APIs, and documentation. Additionally, it offers functionality for code snippet generation, sample project creation, and problem troubleshooting.

{% only_dark %}
![Using Clikkle Assistant's AI features](/clikkle/images/docs/assistant/dark/ask-ai.png)
{% /only_dark %}
{% only_light %}
![Using Clikkle Assistant's AI features](/clikkle/images/docs/assistant/ask-ai.png)
{% /only_light %}

{% info title="Development" %}
While the Clikkle Assistant remains under active development and is considered experimental, it undergoes incremental refinement. Its proficiency in comprehending user queries and delivering relevant responses improves with sustained usage.
{% /info %}

# Getting started  {% #getting-started %}

To engage the Clikkle Assistant, access the Command Center within your Clikkle Console, and proceed to the `Ask the AI` tab in the navigation.

# Querying {% #querying %}

The Clikkle Assistant accommodates queries related to Clikkle, encompassing topics such as:

- How to add platform in the console?
- How to integrate Clikkle in my Flutter project?
- How can I create a user account?
- How to store a file in a bucket?

# Optimizations {% #optimizations %}

For optimal utilization of the Clikkle Assistant, consider the following recommendations:

- **Precision**: Articulate queries with specificity.
- **Natural Language**: The Assistant comprehends plain English, obviating the need for technical terminology.
- **Query Refinement**: In cases of misinterpretation, rephrase queries.
- **Manual Documentation Search**: In scenarios where the Assistant falls short, manually search the Clikkle documentation.

# Privacy {% #privacy %}

When you use the Clikkle Assistant, your questions are sent to OpenAI to generate a response. OpenAI may collect and store your questions and responses for the purposes of improving their services.
