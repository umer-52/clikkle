# UsageFunctions

UsageFunctions

## Properties

| Name | Type | Description |
|------|------|-------------|
| range | string | The time range of the usage stats. |
| executionsTotal | array<metric> | Aggregated stats for number of function executions. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| executionsFailure | array<metric> | Aggregated stats for function execution failures. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| executionsSuccess | array<metric> | Aggregated stats for function execution successes. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| executionsTime | array<metric> | Aggregated stats for function execution duration. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| buildsTotal | array<metric> | Aggregated stats for number of function builds. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| buildsFailure | array<metric> | Aggregated stats for function build failures. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| buildsSuccess | array<metric> | Aggregated stats for function build successes. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| buildsTime | array<metric> | Aggregated stats for function build duration. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
