# StorageUsage

StorageUsage

## Properties

| Name | Type | Description |
|------|------|-------------|
| range | string | The time range of the usage stats. |
| storage | array<metric> | Aggregated stats for the occupied storage size (in bytes). Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| filesCount | array<metric> | Aggregated stats for total number of files. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| bucketsCount | array<metric> | Aggregated stats for total number of buckets. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| bucketsCreate | array<metric> | Aggregated stats for buckets created. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| bucketsRead | array<metric> | Aggregated stats for buckets read. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| bucketsUpdate | array<metric> | Aggregated stats for buckets updated. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| bucketsDelete | array<metric> | Aggregated stats for buckets deleted. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| filesCreate | array<metric> | Aggregated stats for files created. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| filesRead | array<metric> | Aggregated stats for files read. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| filesUpdate | array<metric> | Aggregated stats for files updated. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
| filesDelete | array<metric> | Aggregated stats for files deleted. Can be one of: [Metric model](/docs/references/1.3.x/models/metric) |
