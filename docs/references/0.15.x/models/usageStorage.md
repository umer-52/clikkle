# StorageUsage

| Name | Type | Description |

| range | string | The time range of the usage stats. |
| filesStorage | array<metricList> | Aggregated stats for the occupied storage size by files (in bytes). Can be one of: [Metric List model](/docs/references/0.15.x/models/metricList) |

| bucketsRead | array<metricList> | Aggregated stats for buckets read. Can be one of: [Metric List model](/docs/references/0.15.x/models/metricList) |
| bucketsUpdate | array<metricList> | Aggregated stats for buckets updated. Can be one of: [Metric List model](/docs/references/0.15.x/models/metricList) |
