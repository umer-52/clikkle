# UsageBuckets

UsageBuckets

## Properties

| Name | Type | Description |
|------|------|-------------|
| range | string | The time range of the usage stats. |
| filesCount | array<metricList> | Aggregated stats for total number of files in this bucket. Can be one of: [Metric List model](/docs/references/0.15.x/models/metricList) |
| filesStorage | array<metricList> | Aggregated stats for total storage of files in this bucket. Can be one of: [Metric List model](/docs/references/0.15.x/models/metricList) |
| filesCreate | array<metricList> | Aggregated stats for files created. Can be one of: [Metric List model](/docs/references/0.15.x/models/metricList) |
| filesRead | array<metricList> | Aggregated stats for files read. Can be one of: [Metric List model](/docs/references/0.15.x/models/metricList) |
| filesUpdate | array<metricList> | Aggregated stats for files updated. Can be one of: [Metric List model](/docs/references/0.15.x/models/metricList) |
| filesDelete | array<metricList> | Aggregated stats for files deleted. Can be one of: [Metric List model](/docs/references/0.15.x/models/metricList) |
