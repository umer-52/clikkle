| Name | Type | Description |

| range | string | The time range of the usage stats. |
| documentsCount | array<metricList> | Aggregated stats for total number of documents. Can be one of: [Metric List model](/docs/references/1.0.x/models/metricList) |
| collectionsCount | array<metricList> | Aggregated stats for total number of collections. Can be one of: [Metric List model](/docs/references/1.0.x/models/metricList) |
| documentsCreate | array<metricList> | Aggregated stats for documents created. Can be one of: [Metric List model](/docs/references/1.0.x/models/metricList) |
| documentsRead | array<metricList> | Aggregated stats for documents read. Can be one of: [Metric List model](/docs/references/1.0.x/models/metricList) |
| documentsUpdate | array<metricList> | Aggregated stats for documents updated. Can be one of: [Metric List model](/docs/references/1.0.x/models/metricList) |
| documentsDelete | array<metricList> | Aggregated stats for documents deleted. Can be one of: [Metric List model](/docs/references/1.0.x/models/metricList) |
| collectionsCreate | array<metricList> | Aggregated stats for collections created. Can be one of: [Metric List model](/docs/references/1.0.x/models/metricList) |
| collectionsRead | array<metricList> | Aggregated stats for collections read. Can be one of: [Metric List model](/docs/references/1.0.x/models/metricList) |
