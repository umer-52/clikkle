# UsageBuckets

UsageBuckets

## Properties

| Name | Type | Description |
|------|------|-------------|
| range | string | The time range of the usage stats. |
| filesCount | array<metric> | Aggregated stats for total number of files in this bucket. Can be one of: [Metric model](/docs/references/1.2.x/models/metric) |
| filesStorage | array<metric> | Aggregated stats for total storage of files in this bucket. Can be one of: [Metric model](/docs/references/1.2.x/models/metric) |
| filesCreate | array<metric> | Aggregated stats for files created. Can be one of: [Metric model](/docs/references/1.2.x/models/metric) |
| filesRead | array<metric> | Aggregated stats for files read. Can be one of: [Metric model](/docs/references/1.2.x/models/metric) |
| filesUpdate | array<metric> | Aggregated stats for files updated. Can be one of: [Metric model](/docs/references/1.2.x/models/metric) |
| filesDelete | array<metric> | Aggregated stats for files deleted. Can be one of: [Metric model](/docs/references/1.2.x/models/metric) |
