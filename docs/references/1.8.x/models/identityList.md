# Identities List

Identities List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of identities that matched your query. |
| identities | array<identity> | List of identities. Can be one of: [Identity model](/docs/references/1.8.x/models/identity) |

}
