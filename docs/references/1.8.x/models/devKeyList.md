# Dev Keys List

Dev Keys List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of devKeys that matched your query. |
| devKeys | array<devKey> | List of devKeys. Can be one of: [DevKey model](/docs/references/1.8.x/models/devKey) |

## Example

### REST

```json
{
  "total": 5,
  "devKeys": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "devKeys": ""
}
```
