# Migrations List

Migrations List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of migrations that matched your query. |
| migrations | array<migration> | List of migrations. Can be one of: [Migration model](/docs/references/1.8.x/models/migration) |

## Example

### REST

```json
{
  "total": 5,
  "migrations": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "migrations": ""
}
```
