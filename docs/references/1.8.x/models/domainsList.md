# Domains list

Domains list

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of domains that matched your query. |
| domains | array<domain> | List of domains. Can be one of: [Domain model](/docs/references/1.8.x/models/domain) |

```json
{
  "total": 5,
  "domains": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "domains": ""
}
```
