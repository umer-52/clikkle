# Indexes List

Indexes List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of indexes that matched your query. |
| indexes | array<index> | List of indexes. Can be one of: [Index model](/docs/references/1.8.x/models/index) |

## Example

### REST

```json
{
  "total": 5,
  "indexes": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "indexes": ""
}
```
