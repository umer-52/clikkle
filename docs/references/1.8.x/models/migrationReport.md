# Migration Report

Migration Report

## Properties

| Name | Type | Description |
|------|------|-------------|
| user | integer | Number of users to be migrated. |
| team | integer | Number of teams to be migrated. |
| database | integer | Number of databases to be migrated. |
| row | integer | Number of rows to be migrated. |
| file | integer | Number of files to be migrated. |
| bucket | integer | Number of buckets to be migrated. |
| function | integer | Number of functions to be migrated. |
| size | integer | Size of files to be migrated in mb. |
| version | string | Version of the Appwrite instance to be migrated. |
