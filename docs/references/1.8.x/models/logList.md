# Logs List

Logs List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of logs that matched your query. |
| logs | array<log> | List of logs. Can be one of: [Log model](/docs/references/1.8.x/models/log) |

```json
{
  "total": 5,
  "logs": ""
}
```

### GraphQL

```json
  "logs": ""
```
