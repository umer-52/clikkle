# Executions List

Executions List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of executions that matched your query. |
| executions | array<execution> | List of executions. Can be one of: [Execution model](/docs/references/1.8.x/models/execution) |

{

  "executions": ""
}
