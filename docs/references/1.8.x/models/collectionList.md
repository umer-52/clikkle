# Collections List

Collections List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of collections that matched your query. |
| collections | array<collection> | List of collections. Can be one of: [Collection model](/docs/references/1.8.x/models/collection) |

}
