# Currencies List

Currencies List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of currencies that matched your query. |
| currencies | array<currency> | List of currencies. Can be one of: [Currency model](/docs/references/1.8.x/models/currency) |

## Example

### REST

```json
{
  "total": 5,
  "currencies": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "currencies": ""
}
```
