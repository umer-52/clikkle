# Backup restoration list

Backup restoration list

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of restorations that matched your query. |
| restorations | array<backupRestoration> | List of restorations. Can be one of: [Restoration model](/docs/references/1.8.x/models/backupRestoration) |

## Example

### REST

```json
{
  "total": 5,
  "restorations": ""
}
```

### GraphQL

```json

  "total": 5,

}
```
