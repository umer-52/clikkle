# Metric Breakdown

Metric Breakdown

## Properties

| Name | Type | Description |
|------|------|-------------|
| resourceId | string | Resource ID. |
| name | string | Resource name. |
| value | integer | The value of this metric at the timestamp. |
| estimate | number | The estimated value of this metric at the end of the period. |

## Example

### REST

```json
{
  "resourceId": "5e5ea5c16897e",
  "name": "Documents",
  "value": 1,
  "estimate": 1
}
```

### GraphQL

```json
{
  "resourceId": "5e5ea5c16897e",
  "name": "Documents",
  "value": 1,
  "estimate": 1
}
```
