# Column Indexes List

Column Indexes List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of indexes that matched your query. |
| indexes | array<columnIndex> | List of indexes. Can be one of: [Index model](/docs/references/1.8.x/models/columnIndex) |

## Example

### REST

```json
{
  "total": 5,
  "indexes": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "indexes": ""
}
```
