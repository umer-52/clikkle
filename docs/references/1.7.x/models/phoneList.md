# Phones List

Phones List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of phones documents that matched your query. |
| phones | array<phone> | List of phones. Can be one of: [Phone model](/docs/references/1.7.x/models/phone) |

## Example

### REST

```json
{
  "total": 5,
  "phones": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "phones": ""
}
```
