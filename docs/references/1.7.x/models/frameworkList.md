# Frameworks List

Frameworks List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of frameworks documents that matched your query. |
| frameworks | array<framework> | List of frameworks. Can be one of: [Framework model](/docs/references/1.7.x/models/framework) |

## Example

### REST

```json
{
  "total": 5,
  "frameworks": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "frameworks": ""
}
```
