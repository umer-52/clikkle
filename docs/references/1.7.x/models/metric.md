# Metric

Metric

## Properties

| Name | Type | Description |
|------|------|-------------|
| value | integer | The value of this metric at the timestamp. |
| date | string | The date at which this metric was aggregated in ISO 8601 format. |

## Example

### REST

```json
{
  "value": 1,
  "date": "2020-10-15T06:38:00.000+00:00"
}
```

### GraphQL

```json
{
  "value": 1,
  "date": "2020-10-15T06:38:00.000+00:00"
}
```
