# Health Antivirus

Health Antivirus

## Properties

| Name | Type | Description |
|------|------|-------------|
| version | string | Antivirus version. |
| status | string | Antivirus status. Possible values can are: `disabled`, `offline`, `online` |

## Example

### REST

```json
{
  "version": "1.0.0",
  "status": "online"
}
```

### GraphQL

```json
{
  "version": "1.0.0",
  "status": "online"
}
```
