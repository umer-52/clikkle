# Site Templates List

Site Templates List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of templates documents that matched your query. |
| templates | array<templateSite> | List of templates. Can be one of: [Template Site model](/docs/references/1.7.x/models/templateSite) |

## Example

### REST

```json
{
  "total": 5,
  "templates": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "templates": ""
}
```
