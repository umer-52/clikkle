# Continents List

Continents List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of continents documents that matched your query. |
| continents | array<continent> | List of continents. Can be one of: [Continent model](/docs/references/1.7.x/models/continent) |

## Example

### REST

```json
{
  "total": 5,
  "continents": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "continents": ""
}
```
