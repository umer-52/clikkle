# Variables List

Variables List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of variables documents that matched your query. |
| variables | array<variable> | List of variables. Can be one of: [Variable model](/docs/references/1.7.x/models/variable) |

## Example

### REST

```json
{
  "total": 5,
  "variables": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "variables": ""
}
```
