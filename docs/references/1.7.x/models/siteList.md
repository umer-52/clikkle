# Sites List

Sites List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of sites documents that matched your query. |
| sites | array<site> | List of sites. Can be one of: [Site model](/docs/references/1.7.x/models/site) |

```json
{

  "sites": ""
}
```
