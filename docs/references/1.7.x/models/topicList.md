# Topic list

Topic list

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of topics documents that matched your query. |
| topics | array<topic> | List of topics. Can be one of: [Topic model](/docs/references/1.7.x/models/topic) |

## Example

### REST

```json
{
  "total": 5,
  "topics": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "topics": ""
}
```
