# UsageProject

UsageProject

## Properties

| Name | Type | Description |
|------|------|-------------|
| range | string | The time range of the usage stats. |
| requests | array<metric> | Aggregated stats for number of requests. Can be one of: [Metric model](/docs/references/1.1.x/models/metric) |
| network | array<metric> | Aggregated stats for consumed bandwidth. Can be one of: [Metric model](/docs/references/1.1.x/models/metric) |
| executions | array<metric> | Aggregated stats for function executions. Can be one of: [Metric model](/docs/references/1.1.x/models/metric) |
| documents | array<metric> | Aggregated stats for number of documents. Can be one of: [Metric model](/docs/references/1.1.x/models/metric) |
| databases | array<metric> | Aggregated stats for number of databases. Can be one of: [Metric model](/docs/references/1.1.x/models/metric) |
| users | array<metric> | Aggregated stats for number of users. Can be one of: [Metric model](/docs/references/1.1.x/models/metric) |
| storage | array<metric> | Aggregated stats for the occupied storage size (in bytes). Can be one of: [Metric model](/docs/references/1.1.x/models/metric) |
| buckets | array<metric> | Aggregated stats for number of buckets. Can be one of: [Metric model](/docs/references/1.1.x/models/metric) |
