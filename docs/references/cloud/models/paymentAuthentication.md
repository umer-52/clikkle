# PaymentAuthentication

PaymentAuthentication

## Properties

| Name | Type | Description |
|------|------|-------------|
| message | string | Message for the end user to show on Console. |
| clientSecret | string | Stripe client secret to use for validation. |
| organizationId | string | Organization ID for which the payment authentication is needed. |
| invoiceId | string | Invoice ID against which the payment needs to be validated. |

## Example

### REST

```json
{
  "message": "",
  "clientSecret": "",
  "organizationId": "",
  "invoiceId": ""
}
```
