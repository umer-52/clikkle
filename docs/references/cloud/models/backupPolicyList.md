# Backup policy list

Backup policy list

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of policies that matched your query. |
| policies | array<backupPolicy> | List of policies. Can be one of: [backup model](/docs/references/cloud/models/backupPolicy) |

```json
{
  "total": 5,
  "policies": ""
}
```
