# Mock Number

Mock Number

## Properties

| Name | Type | Description |
|------|------|-------------|
| phone | string | Mock phone number for testing phone authentication. Useful for testing phone authentication without sending an SMS. |
| otp | string | Mock OTP for the number. |

## Example

### REST

```json
{
  "phone": "+1612842323",
  "otp": "123456"
}
```

### GraphQL

```json
{
  "phone": "+1612842323",
  "otp": "123456"
}
```
