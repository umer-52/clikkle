# Runtimes List

Runtimes List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of runtimes that matched your query. |
| runtimes | array<runtime> | List of runtimes. Can be one of: [Runtime model](/docs/references/cloud/models/runtime) |

## Example

### REST

```json
{
  "total": 5,
  "runtimes": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "runtimes": ""
}
```
