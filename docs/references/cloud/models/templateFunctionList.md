# Function Templates List

Function Templates List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of templates that matched your query. |
| templates | array<templateFunction> | List of templates. Can be one of: [Template Function model](/docs/references/cloud/models/templateFunction) |

## Example

### REST

```json
{
  "total": 5,
  "templates": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "templates": ""
}
```
