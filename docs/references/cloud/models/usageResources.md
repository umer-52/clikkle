# UsageResource

UsageResource

## Properties

| Name | Type | Description |
|------|------|-------------|
| name | string | Invoice name |
| value | integer | Invoice value |
| amount | number | Invoice amount |
| rate | number | Invoice rate |
| desc | string | Invoice description |
| resourceId | string | Resource ID |
