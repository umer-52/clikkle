# Functions List

Functions List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of functions that matched your query. |
| functions | array<function> | List of functions. Can be one of: [Function model](/docs/references/cloud/models/function) |

## Example

### REST

```json
{
  "total": 5,
  "functions": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "functions": ""
}
```
