# VCS Content List

VCS Content List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of contents that matched your query. |
| contents | array<vcsContent> | List of contents. Can be one of: [VcsContents model](/docs/references/cloud/models/vcsContent) |

## Example

### REST

```json
{
  "total": 5,
  "contents": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "contents": ""
}
```
