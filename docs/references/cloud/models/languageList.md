# Languages List

Languages List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of languages that matched your query. |
| languages | array<language> | List of languages. Can be one of: [Language model](/docs/references/cloud/models/language) |

## Example

### REST

```json
{
  "total": 5,
  "languages": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "languages": ""
}
```
