# Backup archive list

Backup archive list

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of archives that matched your query. |
| archives | array<backupArchive> | List of archives. Can be one of: [Archive model](/docs/references/cloud/models/backupArchive) |

### REST

```json
{
  "total": 5,
  "archives": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "archives": ""
}
```
