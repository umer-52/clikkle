# Webhooks List

Webhooks List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of webhooks that matched your query. |
| webhooks | array<webhook> | List of webhooks. Can be one of: [Webhook model](/docs/references/cloud/models/webhook) |

## Example

### REST

```json
{
  "total": 5,
  "webhooks": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "webhooks": ""
}
```
