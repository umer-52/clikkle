# Specification

Specification

## Properties

| Name | Type | Description |
|------|------|-------------|
| memory | integer | Memory size in MB. |
| cpus | number | Number of CPUs. |
| enabled | boolean | Is size enabled. |
| slug | string | Size slug. |

## Example

### REST

```json
{
  "memory": 512,
  "cpus": 1,
  "enabled": true,
  "slug": "s-1vcpu-512mb"
}
```

### GraphQL

```json

  "memory": 512,

  "enabled": true,
```
