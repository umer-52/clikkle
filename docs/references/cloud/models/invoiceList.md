# Billing invoices list

Billing invoices list

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of invoices that matched your query. |
| invoices | array<invoice> | List of invoices. Can be one of: [Invoice model](/docs/references/cloud/models/invoice) |

## Example

### REST

```json
{
  "total": 5,
  "invoices": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "invoices": ""
}
```
