# Locale codes list

Locale codes list

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of localeCodes that matched your query. |
| localeCodes | array<localeCode> | List of localeCodes. Can be one of: [LocaleCode model](/docs/references/cloud/models/localeCode) |

## Example

### REST

```json
{
  "total": 5,
  "localeCodes": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "localeCodes": ""
}
```
