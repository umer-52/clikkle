# API Keys List

API Keys List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of keys that matched your query. |
| keys | array<key> | List of keys. Can be one of: [Key model](/docs/references/cloud/models/key) |

## Example

### REST

```json
{
  "total": 5,
  "keys": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "keys": ""
}
```
