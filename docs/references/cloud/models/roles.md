# Roles

Roles

## Properties

| Name | Type | Description |
|------|------|-------------|
| scopes | array | Array of scopes accessible to current user. |
| roles | array | Array of roles assigned to current user. |

## Example

### REST

```json
{
  "scopes": "users.read",
  "roles": "developer"
}
```

### GraphQL

```json
{
  "scopes": "users.read",
  "roles": "developer"
}
```
