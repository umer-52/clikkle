# AdditionalResource

AdditionalResource

## Properties

| Name | Type | Description |
|------|------|-------------|
| name | string | Resource name |
| unit | string | Resource unit |
| currency | string | Price currency |
| price | number | Price |
| value | integer | Resource value |
| invoiceDesc | string | Description on invoice |

## Example

### REST

```json
{
  "name": "",
  "unit": "GB",
  "currency": "USD",
  "price": 5,
  "value": 25,
  "invoiceDesc": ""
}
```

### GraphQL

```json
{
  "name": "",
  "unit": "GB",
  "currency": "USD",
  "price": 5,
  "value": 25,
  "invoiceDesc": ""
}
```
