# DomainSuggestion

DomainSuggestion

## Properties

| Name | Type | Description |
|------|------|-------------|
| domain | string | Domain suggestion. |
| premium | boolean | Is the domain premium? |
| price | number | Domain price. |
| available | boolean | Is the domain available? |

## Example

### REST

```json
{
  "domain": "example.com",
  "premium": true,
  "price": 25.99,
  "available": true
}
```

### GraphQL

```json
{
  "domain": "example.com",
  "premium": true,
  "price": 25.99,
  "available": true
}
```
