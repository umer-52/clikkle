# Payment methods list

Payment methods list

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of paymentMethods that matched your query. |
| paymentMethods | array<paymentMethod> | List of paymentMethods. Can be one of: [paymentMethod model](/docs/references/cloud/models/paymentMethod) |

## Example

### REST

```json
{
  "total": 5,
  "paymentMethods": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "paymentMethods": ""
}
```
