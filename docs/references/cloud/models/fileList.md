# Files List

Files List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of files that matched your query. |
| files | array<file> | List of files. Can be one of: [File model](/docs/references/cloud/models/file) |

## Example

### REST

```json
{
  "total": 5,
  "files": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "files": ""
}
```
