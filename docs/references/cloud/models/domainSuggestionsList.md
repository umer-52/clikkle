# Domain suggestions list

Domain suggestions list

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of suggestions that matched your query. |
| suggestions | array<domainSuggestion> | List of suggestions. Can be one of: [DomainSuggestion model](/docs/references/cloud/models/domainSuggestion) |

## Example

### REST

```json
{
  "total": 5,
  "suggestions": ""
}
```

### GraphQL

```json
{
  "total": 5,
  "suggestions": ""
}
```
