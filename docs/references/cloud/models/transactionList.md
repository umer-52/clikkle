# Transaction List

Transaction List

## Properties

| Name | Type | Description |
|------|------|-------------|
| total | integer | Total number of transactions that matched your query. |
| transactions | array<transaction> | List of transactions. Can be one of: [Transaction model](/docs/references/cloud/models/transaction) |

```json
  "transactions": ""
}
```
