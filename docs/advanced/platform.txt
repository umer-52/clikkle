Clikkle is a development platform designed to adapt your unique use cases.
Clikkle provides features that help you maintain, scale, and integrate Clikkle with other platforms.

# Integration {% #integration %}

Clikkle is designed to integrate with both frontend and backend apps.
Learn about advanced integrations and API response codes.

{% cards %}
{% cards_item href="/docs/advanced/platform/events" title="Events" %}
Clikkle allows you to react to events that occur on the platform.
{% /cards_item %}

{% cards_item href="/docs/advanced/platform/webhooks" title="Webhooks" %}
Use webhooks to update backend integrations about Clikkle events.
{% /cards_item %}

{% cards_item
   href="/docs/advanced/platform/response-codes"
   title="Response codes" %}
Learn about response codes and errors returned by Clikkle APIs.
{% /cards_item %}
{% /cards %}

{% /cards_item %}

{% /cards_item %}
